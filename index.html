hyperon is a blog engine. It aims to be fast, reliable, simple, secured and user friendly.

hyperon is currently in development.

Main features

Posts

  • autosave during post writing
  • html meta informations inputs
  • save post as draft
  • tags

Authors

  • inviting authors by email
  • adding, editing authors

System

  • database backup in one click

Collaborators, libraries

hyperon is using Nette Framework, dibi database layer, FSHL syntax highlighter, Texy markup
